# Feedback 작성 템플릿

`feedback` 스킬이 이 형식으로 `log_for_test/{YYMMDD_HHMM}_{에이전트}_feedback.md`를 만든다. 수동 작성 시 복사해 쓴다.

```
# Feedback — {YYMMDD_HHMM} / {에이전트}

## 유형
bug | enhancement | friction

## 요약
(한 줄)

## 무슨 일이
- 하려던 것:
- 일어난 것:
- 기대한 것:

## 환경
- 에이전트:
- OS:
- 도구:

## 재현
1.
2.

## 관련 스킬·규칙
(예: antithesis, AGENTS.md 9절, resource)
```

> 비밀(.env 값·토큰·자격증명)·프로젝트 작업 내용은 포함하지 않는다. 피드백은 하네스에 관한 것만.
